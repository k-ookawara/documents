■課題２
以下の計算式を読込んで答えを出力するプログラムを作成してください。

```
①
入力：1 + 2
出力：3

②
入力：1 - 2
出力：-1

③
入力：2 * 3
出力：6

④
入力：4 / 2
出力：2

⑤
入力：1 / 3
出力：0.3334
※割り切れない場合は小数点第4位を四捨五入して、それ以降は切り捨ててください。

⑥
入力：150 + 132
出力：282

↓ここからちょっと難しいかも
⑦
入力：1 + -2
出力：-1

⑧
入力：-3 + 1
出力：-2

⑨
入力：-3 + -1
出力：-4

⑩
入力：3 * (12 + 27)
出力：117

⑪
入力：3 * 12 + 27
出力：63

⑫
入力：(3 * 12 + 27)
出力：63

⑬
入力：(8 * (11 + 29) / (7 - 11))
出力：80
```

■課題３
カンマで区切った数値を小さい順に出力してください。

```
①
入力：2,1,5,4,3
出力：1,2,3,4,5

アルファベットをa-zの順で並び替えて出力してください。

②
入力：t,d,b,o,l,q
出力：b,d,l,o,q,t

+-のの順番になるように並び替えて出力してください。
③
入力：-+--+-
出力：++----

④
入力：
-+-+-
++---
-+-++
--+--
+--++

出力：
+++++
+++++
+----
-----
-----
```
